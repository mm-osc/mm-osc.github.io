Red Hat Strengthens Hybrid Cloud's Backbone with Latest Version of Red Hat Enterprise Linux

April 10, 2018 မွာ Red Hat ကေနျပီးမွ ၄င္းတို႕ရဲ႕ Enterprise Linux Platform ျဖစ္တဲ့ Red Hat Enterprise Linux ၏ ေနာက္ဆံုး version ျဖစ္တဲ့ version 7.5 ထြက္ရွိလာျပီ ျဖစ္ေၾကာင္းကို ေၾကျငာခဲ့ပါတယ္။
(Red Hat Enterprise Linux 7.5 ရဲ႕ Beta version ကို Jan 24, 2018 ကထြက္ရွိထားတာျဖစ္ပါတယ္။)

RHEL 7.5 မွာ Security နဲ႔ Compliance အတြက္ Enhanced လုပ္ထားတာေတြ, Storage Costs သက္သာေအာင္ျပဳလုပ္ထားတာေတြ, Microsoft Windows Infrastructure အတြက္ on-premise မွာေရာ Microsoft Azure မွာေရာ Integrated လုပ္ႏိုင္ေအာင္ New function ေတြထည့္ထားတာေတြကို ေတြ႕ရမွာပါ။

RHEL 7.5 မွာ Storage ပုိင္းကိုလည္း Performance ျမွင့္တင့္ထားပါတယ္။RHEL 7.5 မွာ Storage ပိုင္းနဲ႕ပက္သက္ျပီး VDO (Virtual Data Optimizer) ကုိသံုးထားတဲ့အတြက္ေၾကာင့္ data storage costs ေတြကို Cloud နဲ႕ on-premise မွာ ေလွ်ာ့ခ်ေပးႏိုင္မွာျဖ စ္ပါတယ္။VDO က data redundancy ကိုေလွ်ာ့ခ်ေပးႏိုင္ျပီး Storage capacity ကိုတုိးျမွင့္ေပးႏိုင္ပါတယ္။

RHEL 7.5 မွာ Cockpit Administrator Console ကို ျမွင့္တင္မႈေတြလုပ္ထားတဲ့အတြက္ New Linux Systems Administrator ေတြ Troubleshooter ေတြ Developer ေတြအေနနဲ႕ System Management လုပ္ရတဲ့အခါပိုျပီးလြယ္ကူလာမွာျဖစ္ပါတယ္။ Web interface ကလည္းရိုးရွင္းတဲ့အတြက္ management လုပ္တဲ့အခါေတြ, network နဲ႕storage ပုိင္းကို set-ups လုပ္တဲ့အခါေတြမွာ ပိုျပီးလြယ္ကူလာမွာျဖစ္ပါတယ္။

RHEL 7.5 မွာ Windows-based Infrastructure နဲ႕ သံုးလုိ႕ရတဲ့ New function နဲ႕ေပါင္းစပ္မႈေတြပါလာပါတယ္။ Windows Server Implementations နဲ႕ပက္သက္တဲ့ Management ပိုင္းကို Improved လုပ္ထားတာေတြ, Microsoft Azure နဲ႕ Data Transfers ပိုင္းကို ပုိျပီး secure ျဖစ္ေအာင္လုပ္ထားတာေတြ, Microsoft AD နဲ႕ ပက္သက္ျပီး Performance ပိုင္းကို တုိးျမွင့္ထားတာေတြ လုပ္ထားပါတယ္။ Windows-based သံုးထားတဲ့ Organization ေတြအေနနဲ႕ လက္ရွိသံုးထားတဲ့ Windows Infrastructure မွာ RHEL 7.5 ကို ထပ္ျပီး Implement လုပ္တဲ့အခါေတြမွာ ပိုျပီးေတာ့ အဆင္ေျပလြယ္ကူလာေတာ့မွာျဖစ္ပါတယ္။

RHEL 7.5 မွာဆိုရင္ Architecture ေတာ္ေတာ္မ်ား အသံုးျပဳလို႕ရေအာင္ Support ေပးထားပါတယ္။ x86, IBM Power, IBM z Systems နဲ႔ 64-bit Arm ေတြမွာ အသံုးျပဳလုိ႕ရေအာင္ Support ေပးထားပါတယ္။ ဒါ့အျပင္ Single-host KVM virtualization, Open Container Initiative (OCI) runtime environment နဲ႕ base image to IBM z Systems အတြက္ပါ Support လုပ္ေပးထားတာပါတယ္။

Red Hat Enterprise Linux 7.5 ကို ရယူခ်င္တယ္ဆိုရင္ေတာ့ Red Hat Enterprise Linux Subscription ရွိတဲ့ customer တိုင္း Red Hat Customer Portal ကေနသြားေရာက္ Download ရယူလုိ႕ရပါတယ္။

ဘယ္လို Security ပိုင္းေတြျမွင့္တင္ထားတာရွိလည္း, Hybrid Cloud အတြက္ ဘယ္လုိပံ့ပိုုးမႈေတြရွိလည္းဆိုတဲ့ အေသးစိတ္ဖတ္ရႈ႕လုိပါက " https://red.ht/2qmkaY5
